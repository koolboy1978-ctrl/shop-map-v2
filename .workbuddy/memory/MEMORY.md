# 竞品门店地图项目

## 关键文件
- `shop_map.html`: 主地图文件（数据已嵌入代码）
- `竞品店铺信息汇总.xlsx`: 门店数据表格
- `地图标注方案.md`: 标注方案文档
- `sync_map.py`: 同步脚本（备选）
- `check_streets.py`: 街道分类检测脚本
