# 竞品门店地图标注工具

## 需求背景
从 `竞品店铺信息汇总.xlsx` 提取 37 家门店地址，需要在一张地图上标注位置。

## 当前障碍
- **Nominatim（OpenStreetMap）：** 网络无法访问（连接超时）
- **腾讯地图 API：** key `OB4BZ-D4W3U-B7VVO-4PJWW-6TKDJ-W6F7A` 无效（所有接口返回"无效的key"）
- **高德/百度/LocationIQ/Geoapify：** 需要有效 API key
- **浏览器自动化：** MCP 工具未连接，无法通过 Playwright 自动提取

## 解决方案

### 方案 A：手动标注（已实现）
创建交互式 HTML 地图 `shop_map_editor.html`，在浏览器中打开后：
1. 点击地图添加/调整标注点
2. 从左侧列表选择门店，自动关联标注
3. 标注数据自动保存到浏览器 localStorage
4. 导出标注数据 JSON，可生成正式地图

### 方案 B：API Key 方案
在腾讯位置服务（lbs.qq.com）注册免费账号，获取有效 key 后可自动 geocoding。

### 方案 C：近似坐标（不推荐）
基于已知的深圳宝安/光明区各街道大致坐标，用 Leaflet 标注大致区域，但精度较低。
